# フローチャート練習 #6
[Flowchart.dio](./Flowchart.dio)の通りに、入力された値に応じて数字を使った三角形が表示するプログラムを作成してください。

## 実行例
プログラムが完成すると、入力に応じて次のような実行結果が表示されます。

```
n: 3
1
21
321
```

```
n: 5
1
21
321
4321
54321
```

```
n: 9
1
21
321
4321
54321
654321
7654321
87654321
987654321
```

```
n: 10
ERROR
```

```
n: 0
ERROR
```
